# DR SOCIAL SPOT APPLICATION
- Laravel
- VueJS
- Vuetify
- Vuex

## setup the project
- inside `project` folder:
    - copy `.env.example` to `.env`
    - `composer install`
    - `npm install`
    - `php artisan migrate --seed`
    - `npm run dev` (optional)
    - `php artisan serve`
